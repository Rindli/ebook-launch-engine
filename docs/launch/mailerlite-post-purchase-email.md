## Manual MailerLite automation

1. In MailerLite, go to `Automations`.
2. Create a new automation.
3. Choose `When subscriber joins a group`.
4. Select `Quiet Exponential Buyers`.
5. Add one email step using the subject and body above.
6. Set sender name to `Rilind`.
7. Set sender email to `vesesalihumushica@gmail.com`.

This manual step is recommended because MailerLite's public API supports listing and creating draft automations, but the full visual automation setup is still most reliable in the MailerLite UI.
